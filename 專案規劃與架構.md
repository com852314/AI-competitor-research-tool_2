## 一、專案概述

### 任務
追蹤 5 個競品平台的 Slot 遊戲熱門榜與新遊戲，每週執行一次，累積後做跨週趨勢與跨平台分析。

### 5 個調研平台
| 代號 | 全名 | 市場 | 抓取方式 |
|---|---|---|---|
| PT | PT Gaming | 菲律賓 | 截圖 + AI 視覺辨識 |
| BP | BingoPlus | 菲律賓 | HTML 直接 JS 抓 |
| CP | Casino Plus | 菲律賓 | HTML（廠家需從主表查詢）|
| TCG | Luckyi | 菲律賓 | HTML + 名稱視覺辨識首次快取 |
| BAJI | Baji | 孟加拉 | HTML + 名稱視覺辨識首次快取 |

OKB 已從調研範圍移除。

### 每週調研項目
- 各平台 Slot 分類 Top 20 熱門遊戲（含名稱、廠家、排名）
- TCG NEW 分類新遊戲、BAJI New-Old 排序新遊戲

## 五、腳本規格（已實作）

### `capture_week.py`（每週執行）
**輸入**：`data/incoming/raw_input.json`（AI 整理）
**動作**：
1. 從 weekly/ 找最大 W 編號 +1
2. 從 games.json 找主表 ID（用 name+vendor 配對）
3. 新遊戲自動加進 games.json
4. 新廠家報告給用戶（不自動加）
5. 對比上週計算 isNewToTop20、lastWeekRank、rankChange
6. 寫 weekly/W{N}_{date}.json
7. 更新 platforms.json 的 lastCapture

**用法**：
```bash
python scripts/capture_week.py data/incoming/raw_input.json
```

### `viewer.html`（單檔 SPA）
**載入方式**：
- 雙擊或在 Chrome 開啟
- 點「📂 選擇 data 資料夾」按鈕
- （備用）拖檔到頁面

**頁籤**：
1. ✅ 📊 本週快照 - 5 平台 Top 20 並排、★ 跨平台款標示、可點擊展開備註
2. ✅ 🌐 跨平台分析 - ≥3 平台款表格、廠家市占長條圖、自動觀察
3. ✅ 📋 主表瀏覽 - 22+ 款、可搜尋、可按平台數篩選
4. ✅ 🏢 廠家對照 - 26+ 廠家、各平台代碼對照
5. ⏳ 📈 跨週趨勢 - Phase 2D（待累積 ≥2 週資料）

## 六、給未來 AI 的接手指引

### 如果用戶說「我們繼續這個專案」

1. **先讀本文件**（如果你在讀這段，已經做了）
2. **看資料夾結構**：
   - 有 `data/` → 已切換 JSON-first（這是當前狀態 ✓）
   - `data/weekly/` 看有幾週資料（目前只有 W1）
3. **看 docs/週執行SOP.docx v4** 是當週執行步驟的權威來源
4. **不要重做已完成的事**：W1 已完成、主表已建立

### 每週執行的標準流程（SOP v4）

```
0. 用戶登入 5 平台 → 提供分頁
1. AI 抓 PT + BP + CP → 暫存 raw_input.json 的對應欄位
   → 回報辨識失敗清單給用戶 → 用戶確認 → 修正
2. AI 抓 TCG + BAJI（rankings + newGames）→ 補進 raw_input.json
   → 回報辨識失敗、新廠家、跨平台同款建議 → 用戶確認 → 修正
3. AI 執行：python scripts/capture_week.py data/incoming/raw_input.json
   → 自動寫入 weekly/W{N}_*.json + 更新 master
4. AI 回報：新增的主表 ID、發現的新廠家、≥3 平台款摘要、本週觀察
5. 用戶開 viewer.html 看詳細（不用每次生成 Excel）
```

### 用戶想看資料
- **首選**：請用戶開 `scripts/viewer.html`、選 data 資料夾
- **次選**：AI 在對話中直接呈現摘要表格
- **避免**：不再生成 Excel

### 用戶想編資料
- 主表 / 廠家 / 平台對照在 `data/master/*.json`
- 用戶可直接編 JSON，或在對話中告訴 AI 怎麼改
- 改完驗證：`python -m json.tool data/master/games.json`

### Phase 2D：跨週趨勢（待 W2 累積後實作）
等用戶累積 ≥2 週資料後，可加 viewer.html 的「📈 跨週趨勢」頁籤：
- 用 Chart.js 畫排名變動折線圖
- 各廠家市占變化
- 常勝軍 / 曇花一現 統計
- 觸發時機：用戶說「W2 抓完了、做趨勢分析」

## 七、已知問題與注意事項

### Excel ↔ Linux mount 同步（已不適用）
- 切到 JSON 後此問題消失 ✓

### LibreOffice 鎖檔（已不適用）

- 是否優化：未來再說（v5）
